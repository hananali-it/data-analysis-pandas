# Data Analysis with Python and Pandas

This project demonstrates basic data analysis using Python and pandas.

## Project Structure

- `data/sales_data.csv` → sample dataset
- `src/analysis.py` → main analysis script
- `requirements.txt` → project dependencies

## Features

- Reads a CSV dataset
- Calculates average price
- Calculates total quantity sold
- Groups sales by product

## Requirements
